## 前言

日常创建List的机会可太多了,然而老是纠结该怎么创建,尤其是在创建一些一直的比较小的List时.

因此记录一下


#### 方法一

常规方法:

```java
List<Intger> list = new ArrayList<>();
list.add(1);
list.add(2);
list.add(3);
```

#### 方法二

大括号法,不推荐:

```java
List<Integer> list = new ArrayList<>(); 
list.add(1);
list.add(2);
list.add(3);
```
