# ItemCf

物品间的相似如何计算?  基于行为， 喜欢A的人也喜欢B。 那么A和B有相似性.

实现:

(用户对A的兴趣 * A和B的相似性.) + (用户对C端兴趣 * C和B的相似性) 全部加起来得到用户对B的兴趣.
